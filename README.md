# Ejercicios de programación concurrente y sincronización

Este repositorio contiene un conjunto de ejercicios (con soluciones) de programación concurrente y sincronización, tema que se imparte tanto en la asignatura Algoritmos y Estructuras de Datos (ALED) del Grado en Ingeniería Biomédica como en la asignatura Análisis y Diseño de Software (ADSW) del Grado en Ingeniería de Tecnologías y Servicios de Telecomunicación.

- La carpeta raíz contiene un PDF con los enunciados, que aparecen en orden de dificultad creciente.
- La carpeta `Java_Project` contiene un proyecto de Eclipse IDE con las soluciones a todos los ejercicios. Se recomienda **encarecidamente** que el estudiante intente realizar los ejercicios personalmente antes de consultar las soluciones.

El código fuente de los enunciados en LaTeX se ha incluido en la carpeta `LaTeX_Statements`.
